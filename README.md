# PlayerOk Token Checker Pro

## Описание
Многопоточный бот для проверки токенов аккаунтов PlayerOk с улучшенной производительностью и современным дизайном.

## Возможности
- ✅ Проверка единичного токена
- ✅ Многопоточная массовая проверка
- ✅ Подсчет балансов и статистики
- ✅ Современный синий интерфейс
- ✅ Анимации загрузки и прогресс-бар
- ✅ Сохранение результатов в JSON

## Установка
```bash
pip install curl-cffi
```

## Использование
1. Поместите токены в файл `tokens.txt` (по одному на строку)
2. Запустите программу: `python bot.py`
3. Выберите нужный режим из меню

## Получение токена
1. Установите расширение Cookie Editor
2. Авторизуйтесь на playerok.com
3. Скопируйте значение куки `token`
